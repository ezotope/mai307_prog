# Программа для удаления шумов с изображений

## Описание
Программа для тестирования эффективности медианного фильтра при удалении импульсного шума (salt-and-pepper) с изображений в формате PGM.

## Функциональность
- **Медианный фильтр** с тремя размерами окна (3x3, 5x5, 7x7)
- **Три уровня импульсного шума** (5%, 15%, 25%)
- **Метрики качества**: MSE, PSNR, SSIM
- **Автоматическая обработка** всех PGM изображений
- **Экспорт результатов** в CSV файл
- **Русскоязычный интерфейс**

## Тестируемые изображения
- Avia.pgm
- Brigada.pgm  
- Siberia.pgm
- Walk.pgm
- Yakovlev.pgm

## Параметры тестирования
- **Уровни шума**: 5%, 15%, 25% (импульсный шум)
- **Размеры окна**: 3x3, 5x5, 7x7
- **Всего тестов**: 45 (5 изображений × 3 уровня шума × 3 размера окна)

## Метрики оценки качества

### 1. MSE (Mean Squared Error)
- Среднеквадратичная ошибка
- Показывает среднее отклонение пикселей
- Чем меньше значение, тем лучше качество

### 2. PSNR (Peak Signal-to-Noise Ratio)
- Отношение сигнал/шум в децибелах (dB)
- Чем выше значение, тем лучше качество
- Значения выше 30 dB считаются хорошими

### 3. SSIM (Structural Similarity Index)
- Структурное сходство (значения от 0 до 1)
- Наиболее близок к человеческому восприятию качества
- Учитывает яркость, контраст и структуру изображения

## Результаты тестирования

### Лучшие результаты по SSIM:
- Walk.pgm (шум 5%, окно 3x3): SSIM = 0.992944
- Brigada.pgm (шум 5%, окно 3x3): SSIM = 0.988364
- Walk.pgm (шум 5%, окно 5x5): SSIM = 0.983821

### Худшие результаты по SSIM:
- Avia.pgm (шум 25%, окно 3x3): SSIM = 0.744846
- Yakovlev.pgm (шум 25%, окно 3x3): SSIM = 0.742742

## Выводы
1. Медианный фильтр эффективно удаляет импульсный шум
2. Меньшие окна (3x3) лучше сохраняют детали изображения
3. Большие окна (7x7) лучше удаляют шум, но размывают изображение
4. При высоком уровне шума (25%) качество восстановления значительно снижается
5. SSIM показывает наиболее реалистичную оценку качества восстановления

## Технические особенности
- **Язык**: C++
- **Формат изображений**: PGM (Portable GrayMap)
- **Алгоритм**: медианный фильтр
- **Тип шума**: salt-and-pepper (импульсный)
- **Результаты**: CSV файл
- **Интерфейс**: консольное приложение

## Рекомендации
1. Для лучшего качества восстановления использовать окно 3x3
2. При сильном шуме можно попробовать окно 5x5
3. Окно 7x7 использовать только при очень сильном шуме
4. SSIM - наиболее надежная метрика для оценки качества
